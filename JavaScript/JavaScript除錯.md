## 瀏覽器偵錯
`debugger` 除錯語法，程式跑到此語法會停住

- 按下 `Step` 會一行行的跑程式  
（想一次跑完程式就按最左邊藍色的播放鈕）
- 按下 `Watch` 新增想查看的變數，就可以跟著程式看變數的變化
